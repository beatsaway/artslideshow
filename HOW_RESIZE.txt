Image resize to max 700 px (May 2026)
=====================================

Goal
----
Keep slideshow WebP files from being larger than needed for on-screen use:
if either width or height was greater than 700 pixels, the image was scaled
down in proportion so the longest side became 700 pixels (the shorter side
scales accordingly). Files that already fit within 700x700 were left
unchanged (no re-save), to avoid extra quality loss.

Folders processed
-----------------
- sba1
- Op art
- Surrealism
- cubismfaces

Only files named like pasted_images*.webp in each folder were processed.

Tooling
-------
Python 3 with Pillow (PIL), run from the project root. Save the snippet under
"Optional: repeat yourself" as a .py file and execute it.

Algorithm (same as the script)
------------------------------
1. For each pasted_images*.webp in the folders above:
2. Open the image and read width W and height H.
3. If max(W, H) <= 700: skip this file.
4. Otherwise:
   - Convert to RGBA if the source has transparency (RGBA, LA, or palette
     with transparency); otherwise RGB for smaller output.
   - Resize with Pillow's thumbnail((700, 700), LANCZOS). Thumbnail keeps
     aspect ratio and ensures neither side exceeds 700.
   - Close the reader before writing, then overwrite the same .webp path.
   - WebP encoder settings: quality=88, method=6 (same style as earlier
     exports in this project).

Results of the last run (cubismfaces only)
------------------------------------------
- Resized: 112 images (longest side was brought down to 700 px or less).
- Skipped: 75 images (already within 700 px on both dimensions).

Optional: repeat yourself (minimal script)
------------------------------------------
Save as e.g. resize_max_700.py next to this file, install Pillow
(pip install pillow), then run: python resize_max_700.py

  from pathlib import Path
  from PIL import Image

  ROOT = Path(__file__).resolve().parent
  MAX = 700
  for folder in [ROOT / "sba1", ROOT / "Op art", ROOT / "Surrealism", ROOT / "cubismfaces"]:
      for path in sorted(folder.glob("pasted_images*.webp")):
          im = Image.open(path)
          im.load()
          w, h = im.size
          if max(w, h) <= MAX:
              im.close()
              continue
          if im.mode in ("RGBA", "LA") or (
              im.mode == "P" and "transparency" in im.info):
              work = im.convert("RGBA")
          else:
              work = im.convert("RGB")
          im.close()
          work.thumbnail((MAX, MAX), Image.Resampling.LANCZOS)
          work.save(path, "WEBP", quality=88, method=6)
          work.close()

Slideshow HTML files (slideshow.html) did not need changes; they still
reference the same filenames.
